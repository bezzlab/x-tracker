X-Tracker GUI Keyboard Shortcuts

Alt-C Open Configuration Menu

Ctrl-N Create a new X-Tracker configuration

Ctrl-O Open an existing X-Tracker configuration

Ctrl-C Close a X-Tracker configuration

Ctrl-E Exit X-tracker GUI

Alt-H Open Help Menu

Ctrl-H Open help dialog

Ctrl-A Display information about X-Tracker

Alt-S Save X-Tracker configuration

Alt-R Run X-Tracker configuration
