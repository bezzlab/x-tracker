X-Tracker GUI Help

Menu Bar


Configuration Menu

New- Create a new X-Tracker pipeline configuration

Open- Open an existing X-Tracker pipeline configuration

Close- Close a X-Tracker pipeline configuration

Exit- Exit X-tracker GUI


Help Menu

Help Contents- Open this help dialog

About- Show information about X-Tracker


xTracker Configuration Panel

Display the name of the loaded configuration - new or existing

Plugin Selection Panel

Select plugins for each stage of the X-Tracker quantification pipeline, use pull-down lists

Plugin Parameters Panel

Display auto-generated screens for plugin parameter input. Each tab corresponds to the similarly named pull-down list in the Plugin Selection Panel

Run Application Panel

Press Run xTracker to execute the X-Tracker configuration named in the xTracker Configuration Panel

Information Panel

Show information output by X-Tracker during or after it has executed
